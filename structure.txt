===============================================

----------------

- SCSS for styling
- Vanilla JavaScript for interactions
- Daily automated rebuilds for prayer times

----------------

KEY FEATURES
------------

1. PRAYER TIMES SECTION
   - Shia community prayer times
   - Fetched at build time
   - Static HTML for SEO
   - Displays: Fajr, Dhuhr, Asr, Maghrib, Isha

2. CALENDAR DISPLAY
   - Georgian (Western) calendar
   - Islamic Hijri calendar
   - Both dates displayed side-by-side
   - Fetched from API at build time

3. COMMUNITY EVENTS CAROUSEL
   - Banner images only
   - Auto-advance
   - Manual navigation (prev/next)
   - Vanilla JS implementation

4. SERVICES CAROUSEL
   - Similar to events carousel
   - Showcases mosque services
   - Vanilla JS implementation

5. STATIC CONTENT
   - About section
   - Contact information
   - Hero section
   - Footer with important links

SEO OPTIMIZATIONS
-----------------
- Server-side rendering (SSR) at build time
- Semantic HTML5
- Meta tags (Open Graph, Twitter Cards)
- Structured data (JSON-LD for Mosque schema)
- Sitemap.xml auto-generation
- robots.txt configuration
- Optimized images
- Fast loading times (< 1s)
- Perfect Lighthouse scores target

SCSS ARCHITECTURE

- Component-scoped styles
- Mobile-first approach
- BEM naming convention (optional)

BUILD & DEPLOYMENT STRATEGY
---------------------------
- Static site generation (SSG)

- CDN distribution for global performance
- Continuous deployment from Git

PERFORMANCE TARGETS
-------------------
- First Contentful Paint (FCP): < 1s
- Largest Contentful Paint (LCP): < 2.5s
- Total Blocking Time (TBT): < 200ms
- Cumulative Layout Shift (CLS): < 0.1
- Lighthouse Score: 95+/100 (all categories)

BROWSER SUPPORT
